# 📊 Distraction Tracker

A web-based Java Full Stack application that helps users track and manage their daily distractions.
Users can register, log in, add distraction records, view statistics, monitor weekly summaries,
and manage their distraction history through an interactive dashboard.

---

## 🚀 Features

* User Registration & Login Authentication
* Add Daily Distraction Records
* View Total Distraction Time
* View Today's Distraction Time
* View Last 7 Days Statistics
* Weekly Summary Report
* Delete Distraction Records
* Responsive Dashboard UI
* Session Management
* MySQL Database Integration
